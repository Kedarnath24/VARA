# Vara Frontend

React + TypeScript + Vite frontend application.

## Available Scripts

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

### Lint
```bash
npm run lint
```

## Tech Stack

- React 19
- TypeScript
- Vite
- ESLint

## API Configuration

The frontend is configured to communicate with the backend API at `http://localhost:5000`
